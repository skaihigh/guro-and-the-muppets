# Vårherres Klinkekule - Swing Jazz Arrangement

**Original**: Erik Bye (tekst) / Finn Luth (melodi)
**Arrangement**: Guro and the Muppets
**Key**: E minor (original toneart)
**Tempo**: 130-140 BPM (medium swing)

## Quick Reference

### Toneart
- **Concert pitch**: E minor
- **Tenor sax**: F#m (skrevet)
- **Alto sax**: C#m (skrevet)

### Form
```
INTRO → VERS 1 → VERS 2 → MELLOMSPILL → VERS 3 → VERS 4 →
[GENE KRUPA BREAK] → VERS 5 → OUTRO
```

### Akkordprogresjon (Vers, reharmonisert)
```
| Em9 | Bm7 E7 | Am9 | D13 |
| Gmaj7 | G#dim7 | Am7 | D7 |
```
