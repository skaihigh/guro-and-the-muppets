## SHELL VOICINGS (Venstre hånd)

```
Gmaj7:   G + F# (root + maj7)
Gmaj9:   G + F# + A (root + maj7 + 9th)
G6:      G + E (root + 6th)
Cmaj7:   C + B (root + maj7)
Bm7:     B + A (root + 7th)
E7:      E + D (root + 7th)
Am7:     A + G (root + 7th)
Am9:     A + G + B (root + 7th + 9th)
D7:      D + C (root + 7th)
D13:     D + C + B (root + 7th + 13th)
Em7:     E + D (root + 7th)
A7:      A + G (root + 7th)
C#dim7:  C# + E + G + Bb (passing chord)
E7#9:    E + D + G# (root + 7th + #9)
```

## TIPS

- C#dim7 er klassisk passing chord mellom C og D!
- Chorus: Rhythm changes feel - mer energi
- Bridge: Følg dynamikken, bygg mot klimaks
- Hold det SPARSE under vokal
