# Crescendo i gågata - RYTMEGITAR

**Toneart**: G major | **Tempo**: 140-150 BPM | **Feel**: Medium-Up Swing

---

## AKKORDPROGRESJON

### Vers:
```
| Cmaj7 | Bm7 E7 | Am9 | D13 |
| Em7 A7 | Am7 D7 | Gmaj9 | G6 |
```

### Chorus:
```
| Cmaj7 C#dim7 | Gmaj7/D G6 |
```

---

## SHELL VOICINGS (Beginner-friendly!)

### Standard grep:
```
Gmaj7:  3 x 4 4 x x   (eller: x x 5 4 3 x)
G6:     3 x 2 4 x x   (eller: 3 2 0 0 0 0)
Cmaj7:  x 3 2 4 x x   (eller: 8 x 9 9 x x)
Am7:    x 0 2 0 1 0   (åpen!)
D7:     x x 0 2 1 2   (åpen!)
Bm7:    x 2 4 2 3 x
E7:     0 2 0 1 0 0   (åpen!)
Em7:    0 2 2 0 3 0   (åpen!)
C#dim:  x 4 5 3 5 x   (passing)
```

---

## CAPO ALTERNATIV: Capo 5. bånd

Med capo 5 spiller du i D-former:
```
G  = D-form:   x x 0 2 3 2
C  = G-form:   3 2 0 0 0 3
Am = Em-form:  0 2 2 0 0 0
D7 = A7-form:  x 0 2 0 2 0
Em = Bm-form:  x 2 4 4 3 2 (eller barre)
```

---

## STRUMMING

### Fire på gulvet:
```
| ↓  ↓  ↓  ↓ |
  choo CHIT choo CHIT
      ↑        ↑
   mute/aksent
```

### Teknikk:
- Slag 1 og 3: Lett
- Slag 2 og 4: Mute/aksent
- Match hi-hat!

## FORM MED DYNAMIKK

| Del | Dynamikk | Hva du gjør |
|-----|----------|-------------|
| Intro | mp | Lett eller vente |
| Vers 1-2 | mf | Fire på gulvet |
| Chorus | f | Litt mer energi! |
| Vers 3-4 | mf | Steady |
| Bridge | mp→f | Følg dynamikken |
| Vers 5 | ff | Full strumming |
| Outro | f→mp | Ned igjen |

---

## TIPS

1. **G-dur er gitarvennlig** - mange åpne grep!
2. **Chorus**: Kan gi litt mer
3. **C#dim** = passing chord, bare et slag
4. **Fire på gulvet**, MYKT
5. **Lås deg på hi-hat**
6. Ikke spill for høyt!
