# Vårherres Klinkekule - RYTMEGITAR

**Toneart**: E minor | **Tempo**: 130-140 BPM | **Feel**: Medium Swing

---

## AKKORDPROGRESJON (Vers)

```
| Em9 | Bm7 E7 | Am9 | D13 |
| Gmaj7 | G#dim7 | Am7 | D7 |
```

---

## SHELL VOICINGS (3 noter - Freddie Green stil)

### Åpne grep (gitarvennlig!):

```
Em6:    0 2 2 0 2 0   (åpen - veldig enkelt!)
Em7:    0 2 2 0 3 0   (åpen)

Am7:    x 0 2 0 1 0   (åpen)
Am9:    x 0 2 2 1 0   (åpen)

D7:     x x 0 2 1 2   (åpen)
D9:     x x 0 2 1 0   (åpen - legg til 9)

Gmaj7:  3 2 0 0 0 2   (åpen)
G6:     3 2 0 0 0 0   (åpen)

Bm7:    x 2 4 2 3 2   (barre, eller:)
        x 2 0 2 0 2   (forenklet)

B7:     x 2 1 2 0 2   (åpen)

E7:     0 2 0 1 0 0   (åpen)

G#dim7: x x 6 7 6 7   (eller 4 x 3 4 3 x)
```

---

## STRUMMING PATTERN

### Basic (fire på gulvet):
```
| ↓  ↓  ↓  ↓ |
  choo CHIT choo CHIT
      ↑        ↑
   aksent   aksent (2 og 4)
```

### Teknikk:
- Slag 1 og 3: Lett, åpen streng
- Slag 2 og 4: Mute/choke, aksent
- Lytt til hi-hat - match den!

---

## FORM MED DYNAMIKK

| Del | Dynamikk | Hva du gjør |
|-----|----------|-------------|
| Intro | mp | Kan vente, eller veldig lett |
| Vers 1-2 | mf | Steady fire på gulvet |
| Mellomspill | mf | Hold groove |
| Vers 3 | mp | Lettere, kan pause |
| Vers 4 | mf→f | Bygg med bandet |
| Break | - | STOPP! Kun trommer |
| Vers 5 start | p | Vente eller nesten uhørlig |
| Vers 5 slutt | f | Full strumming |
| Outro | f→p | Følg dynamikken ned |

---

## TIPS

1. **E minor er PERFEKT for gitar** - masse åpne grep!
2. **Færre noter er bedre** - 3-note voicings er nok
3. **Mute på 2 og 4** - det gir "chick" lyden
4. **Lytt til trommene** - lås deg på hi-hat
5. **Ikke spill for høyt** - du er lim, ikke solist
6. **Gene Krupa break**: STOPP helt - kun trommer!

### Du er den viktige linken mellom trommer og bass!

---

## VANLIGE FEIL Å UNNGÅ

- Spille for høyt
- For mange noter i akkordene
- Glemme å aksente 2 og 4
- Spille over vokalisten
- Ikke lytte til resten av bandet
