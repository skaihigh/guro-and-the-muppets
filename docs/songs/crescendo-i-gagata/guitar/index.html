Guitar

From: Crescendo i Gagata

Crescendo i gågata - RYTMEGITAR

Toneart: G major (vers) / E minor (chorus) | Tempo: 130-140 BPM | Feel: Medium Swing

TONEART-KONSEPT

  • Vers: G-dur (åpne grep, glad)
  • Chorus: E-moll (Ode to Joy, dramatisk)
  • Pivot: B7 på slutten av vers → Em

AKKORDPROGRESJON

Intro (E-moll):

| Em9 | Am7 D7 | Em9 | B7#9 |

Vers (G-dur):

| Cmaj7 | Bm7 E7 | Am9 | D13 | Em7 A7 | Am7 D7 | Gmaj9 | G6 B7 |

Chorus - Ode to Joy (E-moll):

| Em9 | Am7 D7 | Gmaj7 | B7#9 | Em9 | Am7 D7 | Gmaj7 C#m7b5 | F#7 B7 |

SHELL VOICINGS

Vers (G-dur, åpne grep - gitarvennlig!):

Cmaj7:  x 3 2 0 0 0   (åpen)
Bm7:    x 2 4 2 3 2   (barre)
E7:     0 2 0 1 0 0   (åpen)
Am7:    x 0 2 0 1 0   (åpen)
Am9:    x 0 2 2 1 0   (åpen)
D7:     x x 0 2 1 2   (åpen)
D13:    x x 0 2 1 0   (åpen variant)
Em7:    0 2 2 0 3 0   (åpen)
A7:     x 0 2 0 2 0   (åpen)
Gmaj7:  3 2 0 0 0 2   (åpen)
Gmaj9:  3 x 0 2 0 2
G6:     3 2 0 0 0 0   (åpen)

Chorus (E-moll, også mange åpne grep!):

Em9:    0 2 0 0 0 2   (åpen)
Am7:    x 0 2 0 1 0   (åpen)
D7:     x x 0 2 1 2   (åpen)
Gmaj7:  3 2 0 0 0 2   (åpen)
B7:     x 2 1 2 0 2   (åpen)
B7#9:   x 2 1 2 0 3   (alternativ)
C#m7b5: x 4 2 4 0 0   (la piano ta denne hvis vanskelig)
F#7:    2 4 2 3 2 2   (barre)
Em6:    0 2 2 0 2 0   (åpen)

STRUMMING

Basic (fire på gulvet):

| ↓  ↓  ↓  ↓ |
  choo CHIT choo CHIT
      ↑        ↑
   mute/aksent (2 og 4)

Teknikk:

  • Slag 1 og 3: Lett, åpen streng
  • Slag 2 og 4: Mute/choke, aksent
  • Match hi-hat!

Vers vs Chorus:

  • Vers: Lett, glad strumming
  • Chorus: Kan være litt mer aggressiv (Ode to Joy dramatikk)

SPESIELLE STEDER

Pivot (vers → chorus):

Takt 8: | G6    B7   | → | Em9 ...
            ↑
      pivot chord - gi den vekt!

Intro:

| Em9 | Am7 D7 | Em9 | B7#9 |

Mørk stemning - kan spille arpeggio eller bare holde akkorder

FORM MED DYNAMIKK

Del Dynamikk Toneart Hva du gjør
Intro mp Em Lett eller vente
Vers 1-2 mf G Fire på gulvet, glad
Chorus 1 f Em Litt mer energi!
Vers 3-4 mf G Steady
Chorus 2 f Em Enda mer
Bridge start mp G Ta ned
Bridge slutt f →Em Bygg!
Sax solo mf Em Stødig groove
Vers 5 ff G Full strumming
Chorus 3 f→mp Em Rit. på slutten

TIPS

  1. G-dur OG E-moll er gitarvennlige - masse åpne grep!
  2. Kontrast: Vers lyst, chorus mørkere
  3. B7 er pivot - gir fin overgang G → Em
  4. C#m7b5 er vanskelig - la piano ta den, eller skip
  5. Fire på gulvet, MYKT
  6. Lås deg på hi-hat - dere er 2&4-teamet!
  7. Ikke spill for høyt under vokal

CAPO ALTERNATIV

Med capo på 2. bånd kan du spille i D-former over Em:

Em = Dm-form:  x x 0 2 3 1
Am = Gm-form:  3 5 5 3 3 3
B7 = A7-form:  x 0 2 0 2 0

Men åpne grep fungerer utmerket uten capo!
